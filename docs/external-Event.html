    preventDefault() → {undefined}

    Tells that if the event does not get explicitly handled, its default action should not be taken as it normally would be. The event continues to propagate as usual, unless one of its event listeners calls external:Event#stopPropagation or external:Event#stopImmediatePropagation, either of which terminates propagation at once.

    stopImmediatePropagation() → {undefined}

    Prevents other listeners of the same event from being called. If several listeners are attached to the same element for the same event type, they are called in the order in which they were added. If invoked during one such call, no remaining listeners will be called.
