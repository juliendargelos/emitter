### Install

With yarn:

```shell
yarn add @juliendargelos/emitter
```

With npm:

```shell
npm install @juliendargelos/emitter --save
```

### Development

Clone the repository:

```shell
git clone git@github.com:juliendargelos/emitter.git
```

Install dependencies:

```shell
yarn
```

Link git hooks:

```shell
yarn hooks:link
```

Generate documentation:

```shell
yarn docs
```

Serve documentation:

```shell
yarn docs:serve
```

Lint:

```shell
yarn lint
```

Test:

```shell
yarn test
```

Build:

```shell
yarn build
```
